# proj-data
> Projection Data Compressed to Fit in the Browser

# features
- fits in the browser
- loads super quickly
- includes PROJ4 Strings, ESRI WKT, and OGC WKT
- data scraped from epsg.io

# install
```bash
npm install proj-data
```

# usage
```js
import data from "proj-data";

console.log(data["EPSG:32617"]);
```
